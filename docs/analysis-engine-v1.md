# Analysis Engine v1

## Призначення

Перша версія створює прозорий детермінований прогноз для одного активу Pocket Option і
експірації 1, 2 або 3 хвилини. Вона не відкриває угод і не використовує прихований AI.

Офіційним джерелом даних є тільки Pocket: остання котировка, тики, 30-секундні, M1 та M5
свічки. OTC і Regular мають окремі ваги та окремі версії алгоритму.

## Що впливає на напрям

- EMA 9, EMA 20 та EMA 21;
- локальний нахил і прискорення;
- тіло, тіні та серія свічок;
- ATR і режим волатильності;
- підтримка, опір, положення в діапазоні;
- пробій, хибний пробій, відбій, імпульс і виснаження;
- рух, швидкість, баланс напрямків і розворот останніх тиків;
- M5 лише як слабший локальний контекст;
- свіжість і повнота Pocket-даних.

Виплата не входить до ознак, ваг, напряму, сили, пояснення або ризику. Backend повертає
її лише як інформаційне поле для інтерфейсу.

## Технічні обмеження

Напрям не створюється, якщо:

- Pocket не підключений або не авторизований;
- актив недоступний;
- котировка відсутня, пошкоджена або старша за 15 секунд;
- немає щонайменше 21 завершеної M1-свічки;
- немає щонайменше 12 завершених 30-секундних свічок;
- за останні 30 секунд отримано менше 8 тиків.

Оцінка `0–100` є силою узгодженості математичних ознак, а не заявленою ймовірністю
виграшу. Активні версії: `market-pulse-deterministic-otc-v1.0.0` та
`market-pulse-deterministic-regular-v1.0.0`.

## API

`POST /api/analyze` вимагає справжній Telegram `initData` у заголовку
`X-Telegram-Init-Data`.

```json
{
  "assetId": "UUID вибраного активу",
  "expirationMinutes": 1
}
```

Endpoint не сканує каталог, не запускає аналіз інших активів і не повертає великі
масиви сирих свічок або тиків.

## Наступний етап

Ця версія показує дослідницький прогноз. Серверна фіксація фактичної ціни входу,
таймер експірації, фінальна котировка, WIN/LOSS/DRAW/INVALID/CANCELLED та запис прогнозу
в Supabase реалізуються окремо в `trade-timer-v1`. Тіньове навчання не змінює правила
цієї версії й буде додане після накопичення якісних завершених результатів.
